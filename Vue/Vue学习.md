# Vue学习 #

- 安装Node.js环境，安装好之后，对安装环境进行验证

  - 查看node版本

    ```shell
    node -v
    ```

    版本号：

    ```shell
    v10.15.3
    ```

  - 查看npm版本

    ```shell
    npm -v
    ```

    版本号：

    ```shell
    6.4.1
    ```

- 通过npm全局安装vue-cli

  ```shell
  sudo npm install --global vue-cli
  ```

  - 查看vue版本

    ```shell
    vue --version
    ```

    版本号：

    ```shell
    2.9.6
    ```

  如果能显示版本号，说明`vue-cli`命令已经安装成功
